MCProject - startup smart contract development environment

EBMobile Technique
2018-06-18

peer节点主机

ssh sopaco@47.105.106.250

环境说明

合约路径：/home/config/chaincode/go/${project_contract}
镜像YAML配置：/home/bc/docker-peer0org1.yaml

进入docker

docker exec -it cli bash

合约文件共享映射

${HOME_ROOT}/config/chaincode/go/ –> /opt/gopath/src/github.com/hyperledger/fabric/chaincode/go

安装并实例化智能合约

peer chaincode install -n practise2018 -v 1.0 -p github.com/hyperledger/fabric/chaincode/go/practise2018
peer chaincode instantiate -n practise2018 -v 1.0 -c ‘{“Args”:[]}’ -C bc

添加智能合约关联到java注册服务

http://47.105.109.237:8080/chaincode/add
